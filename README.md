## ❓ If something goes wrong

If the app does not open:

- Check that the file finished downloading

- Right-click the file and choose Run as administrator
- Make sure Windows did not block it
- Try opening it from a simple folder path like `C:\ClaudeCode`

- Wait a few seconds
- Press Enter
- Restart the app
- Close other terminal windows
- Check that your internet connection works

If sign-in fails:

- Recheck your account details
- Paste the API key again
- Confirm the key has access
- Make sure the copied text has no spaces before or after it

If file changes do not appear:

- Refresh the folder view
- Open the file again in your editor
- Check whether the app saved changes to a different copy
